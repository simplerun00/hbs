PDF를 서버에서 JPG ZIP으로 변환

PDF 파일을 백엔드 서버로 보내면 서버가 페이지별 JPG를 만들고 ZIP 파일로 묶어 돌려줍니다. 큰 파일도 브라우저가 직접 렌더링하지 않아 기존보다 더 안정적으로 처리할 수 있습니다.

서버 변환

1. 변환 설정

백엔드 주소를 넣고 PDF를 선택하면 서버 변환을 시작합니다.

사용 방법

  • 백엔드 주소를 한 번 입력하면 브라우저에 저장됩니다.
  • PDF를 선택하고 필요하면 페이지 범위를 입력합니다.
  • `서버에서 ZIP 만들기`를 누르면 서버가 JPG 묶음을 만들어 줍니다.
  • 완료되면 ZIP 파일이 자동 다운로드되고 다시 받기 버튼도 활성화됩니다.

예시 입력

전체 / 1-3 / 2,4,6 / 1-2,5

Result
