PDF를 이미지로 바로 변환

PDF 파일을 브라우저에서 열어 페이지별 JPG 이미지로 저장합니다. 파일은 서버로 보내지지 않고 현재 브라우저 안에서만 처리됩니다.

브라우저 처리

1. 파일 선택

PDF를 선택하면 변환 준비를 시작합니다.

사용 방법

  • PDF를 하나 선택합니다.
  • 필요하면 페이지 범위를 입력합니다. 비워두면 전체 페이지를 처리합니다.
  • `이미지 만들기`를 누르면 미리보기가 생성됩니다.
  • 각 페이지별로 JPG 다운로드하거나 `전체 다운로드`를 누릅니다.

예시 입력

전체 / 1-3 / 2,4,6 / 1-2,5
